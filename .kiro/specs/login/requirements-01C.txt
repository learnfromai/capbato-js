Form Submission:
WHEN a user clicks the "Login" button
THE SYSTEM SHALL prevent the default form submission and trigger authentication process

WHEN a user presses Enter key in any form field
THE SYSTEM SHALL trigger form submission and proceed with authentication

WHEN the login form is submitted
THE SYSTEM SHALL extract username and password values from the input fields

WHEN the login form is submitted with empty username field
THE SYSTEM SHALL proceed with authentication attempt (server will handle validation)

WHEN the login form is submitted with empty password field
THE SYSTEM SHALL proceed with authentication attempt (server will handle validation)
